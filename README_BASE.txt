# Gestor de pedidos - Proyecto base para P10

Este proyecto está preparado para la práctica P10 de Git, GitHub, refactorización, documentación y CI.

Debe funcionar ejecutando:

python app.py

El proyecto contiene problemas intencionados de diseño y mantenibilidad para que el alumnado pueda analizarlos y refactorizarlos durante la práctica.

No incluye tests, configuración de Ruff, workflow de GitHub Actions ni documentación completa. Todo eso debe añadirse como parte de los ejercicios.
